# Introduction

This library was created thanks to the [Next International](https://github.com/QuiiBz/next-international) project and its beautiful internal international-types package, which offers developers strong typing for translations. A huge thank you to [Tom Lienard](https://x.com/tomlienard).

The goal is to offer a simple, platform-based API so that typesafe translations can be used without any middleware or extra configuration, but rather with an identical API on the server and client sides to make everything cleaner in the developer's day-to-day work.

## Features

- 100% Type-safe: Type-safe translations
- Platform-based API: Works seamlessly with Next.js, React Router, and TanStack Router
- No extra configuration: No middleware or providers
- Identical API on the server and client sides: Use the same API on the server and client sides. No hooks on one side and functions on the other with different names.
